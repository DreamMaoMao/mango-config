# Usage
```bash
git clone https://github.com/Dreammango/mango-config.git ~/.config/mango
```
## Some Common Default Keybindings

- alt+return: open foot terminal
- alt+space: open rofi launcher
- alt+q: kill client
- alt+left/right/up/down: focus direction
- super+m: quit mango
